## Instruções de Uso

1. Copie o template acima
2. Preencha o frontmatter YAML com as informações do artigo
3. Substitua os placeholders [entre colchetes] pelo conteúdo real
4. Adicione/remova seções conforme necessário (mínimo 5 seções)
5. Salve como `nome-do-slug.md` na pasta `content/blog/`
6. Atualize o arquivo `src/lib/blog.ts` com os metadados

## Checklist Rápido

- [ ] Frontmatter completo e válido
- [ ] Título otimizado (50-70 caracteres)
- [ ] Meta description (150-160 caracteres)
- [ ] Sumário com links âncora
- [ ] Mínimo 5 seções H2
- [ ] Pelo menos 1 tabela
- [ ] Pelo menos 2 listas
- [ ] Exemplos práticos
- [ ] CTAs distribuídos
- [ ] Posts relacionados
